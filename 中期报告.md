1. 文献调研（第一阶段，已完成）
（1）系统学习了 Chern 平带模型、分数陈绝缘体（FCI）的基本理论框架；

（3）学习了广义泡利原理、Jack 多项式以及手征 Luttinger 液体理论关于 FCI 边缘激发的预言；
（4）重点研读何爱垒博士论文中关于碟形几何 Kagome FCI 边缘谱的章节，明确了基态角动量、准简并序列、磁通演化等可对比的数值判据。

2. 代码框架与碟形格子构造（第二阶段，已完成）
（1）在 Python + NumPy/SciPy + QuSpin 环境下搭建完整数值代码框架，模块化为格子构造、单体哈密顿量、C₆ 对称性、严格对角化、绘图等部分；
（2）碟形格子构造：从 Kagome 周期格子的 [-10, 10] × [-10, 10] 平铺出发，以六角空腔中心为原点，按距离排序选取最近的 72 个格点。验证了 C₆ 旋转对称性（构建置换 σ，验证 σ⁶ = 恒等映射，最大误差 < 10⁻⁸），格点总数严格等于 72 = 24 × 3，且 A、B、C 三个子格各占 24 个；
（3）多体 ED 框架基于 QuSpin 的 boson_basis_general，利用 C₆ 旋转对称性把 Hilbert 空间按角动量量子数分块对角化（六个扇区），将 N_b = 4 时的 Hilbert 空间维度从约 10⁶ 降至每扇区约 1.7 × 10⁵。

3. 单体哈密顿量与单粒子能谱（第二阶段，已完成）
（1）单粒子哈密顿量：包含最近邻 t = 1 跃迁（带 Peierls 相位 φ = 0.22π，规约为 A→B、B→C、C→A 取 +φ）、次近邻 t' = −0.19 跃迁，以及 V_trap = 0.005 的谐振势（以 a/2 为长度单位）；
（2）单体能谱：直接对角化 72 × 72 矩阵，按 C₆ 角动量 L mod 6 分类，复现了何爱垒论文图 3.4 的结构——基态 L = 5，低能轨道按 L = 5, 4, 3, 2, 1, 0, 5, 4, … 顺序排布，与文献完全一致，验证了单粒子模型构建的正确性。

4. 多体严格对角化（V_nn = 0），复现骆某论文图（第二阶段，已完成）
（1）完成了 N_b = 2、3、4、5 共四种粒子数下 V_nn = 0 时的多体严格对角化，得到对应的低能能谱及 C₆ 角动量量子数标记；
（2）N_b = 3、4、5 的能量–角动量谱与骆某博士论文中相应的多体能谱图（N_b = 3, 4, 5, 6 四面板）逐图一一对应：基态角动量、低能流形的能级排布、能隙大小都与文献一致，从而验证了多体哈密顿量构造与 C₆ 分块对角化实现的正确性；
（3）以 N_b = 4 为例，V_nn = 0 时基态出现在 L = 2 扇区，能量 −8.7266，与最低激发态能隙约 0.008，符合 FCI 准简并基态结构。

5. V_nn 与 V_nnn = V_nn/3 的初步扫描（第三阶段，初步尝试）
作为相变研究的初步探索，初步尝试了两类扫描：（1）纯 NN 排斥扫描，包括 N_b = 3 在 V_nn ∈ [0, 5] 区间（26 个参数点）和 N_b = 4 在 V_nn ∈ [0, 20] 区间（21 个参数点）；（2）V_nnn = V_nn/3 扫描，对 N_b = 4 在 V_nn ∈ [0, 20] 区间扫描 21 个参数点。每个参数点在六个 C₆ 扇区分别求出最低 20–30 个本征态，扫描结果以 NPZ 格式增量保存，避免长时计算意外中断造成的损失。

初步观察：
（1）N_b = 4 纯 NN 扫描中，基态角动量随 V_nn 发生明显跳变：V_nn = 0 → L = 2，V_nn ≈ 1 → L = 3，V_nn ≥ 2 → L = 0；
（2）V_nnn = V_nn/3 扫描中，基态角动量演化为 L = 2 → 0 → 4 → 2 → …，路径与纯 NN 排斥不同；
（3）N_b = 3 在 V_nn ∈ [0, 5] 区间也出现了清晰的基态角动量跳变。

值得注意的是，虽然各扫描中均观察到基态角动量随相互作用强度的明显移动，但低能谱中尚未出现新的、与 ν = 1/4 相对应的准简并序列——这提示在已有扫描区间内或许尚未完全进入 ν = 1/4 相，需要进一步加密扫描点、扩大扫描范围、并尝试更多扫描方案，作为下一阶段的重点工作。
